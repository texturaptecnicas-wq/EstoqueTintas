
# ERP Estoque de Tintas

Sistema completo de gestão de estoque de tintas com alertas automáticos, integração WhatsApp e importação de dados via Excel/CSV.

## Funcionalidades

- ✅ Cadastro manual de produtos (tintas)
- ✅ Importação em massa via Excel/CSV com mapeamento de colunas
- ✅ Alertas automáticos de estoque baixo
- ✅ Integração com WhatsApp para pedidos rápidos
- ✅ Interface otimizada para usuários idosos (fontes grandes, alto contraste)
- ✅ Tabela responsiva com scroll horizontal touch-friendly
- ✅ Destaque visual de produtos com estoque crítico (vermelho/amarelo)
